# Активности — iPhone (без App Store)

Это **PWA** (Progressive Web App): ставится на домашний экран iPhone через Safari и работает **полностью offline**.

## Установка на iPhone

1. Скопируйте папку `ShturvalVozhatogo-iOS-PWA` на Mac.
2. Откройте в ней локальный сервер (Safari не любит `file://` для SW):

```bash
cd ~/Desktop/ShturvalVozhatogo-iOS-PWA
python3 -m http.server 8080
```

3. На iPhone и Mac — **одна Wi‑Fi сеть**.
4. В Safari на iPhone откройте: `http://<IP-вашего-Mac>:8080`
   (IP: Системные настройки → Сеть, например `http://192.168.1.10:8080`).
5. Кнопка **Поделиться** → **«На экран „Домой“»** → Добавить.
6. Запускайте с иконки «Штурвал» — как обычное приложение.

### Если нужен HTTPS / постоянный доступ

- Залейте папку на любой статический хостинг (GitHub Pages, Netlify, Cloudflare Pages).
- Откройте URL в Safari → «На экран Домой».

## Что умеет

- 100 игр, поиск, пресеты, категории  
- Карточка + **режим ведущего** (шаги, таймер, оценка)  
- Избранное и история (localStorage)  
- **Подборка на N дней** с правилом «одно место на день»
